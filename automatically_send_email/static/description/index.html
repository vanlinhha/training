Automatically send email to users when they sign up successfully

Before using this module, allow your user to sign up by go to menu Setting/ General setting, check the line "Allow external user to sign up"

Next, activate the develop mode for create a mail server

Go to menu Setting/Email/Outging Mail Server/ and press button Create to create a outgoing mail server

Fill in the form your server information, press Save and Test Connection
a popup like this will appear when you create successfully your server

Everything is ready, you can use this module now.
